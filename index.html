🔗 Conectar con API de resultados

1
Crea cuenta gratis en football-data.org y copia tu API key desde tu perfil.
2
Entra como admin → pestaña Partidos → pega la API key y guarda.
3
Importante: Los navegadores bloquean llamadas directas (CORS). Necesitas un proxy en Netlify o Vercel (gratis). Crea el archivo netlify/functions/proxy.js:
exports.handler = async (event) => {
  const key = process.env.FOOTBALL_API_KEY;
  const url = event.queryStringParameters.url;
  const res = await fetch(url, {headers:{'X-Auth-Token':key}});
  const data = await res.json();
  return {statusCode:200, body:JSON.stringify(data),
    headers:{'Access-Control-Allow-Origin':'*'}};
};
4
En Netlify agrega la variable de entorno FOOTBALL_API_KEY con tu key. Listo — el botón "Importar de API" funcionará automáticamente.
💡 Si migras de GitHub Pages a Netlify (también gratis), el proxy queda integrado. Dime y te explico cómo migrar.
